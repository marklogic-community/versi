Versi Getting Started Guide

What is Versi?

The Versi template is an HTML design template whose goal is to increase the productivity of developers when building or rapid prototyping applications. Because designing and wiring up a user interface is one of the most time consuming parts of the development cycle, Versi includes pre-wired XQuery example pages, UI modules, and widgets that can be customized to suit your application.

 

Versi also includes a host of features to begin wiring up AJAX applications without the tedious setup process.

Features include:

  • Modular design
  • Sciptaculous Javascript framework
  • Global Javascript library
  • Design widgets
  • Prewired XQuery pages with supporting modules

 

For full documentation on Versi please see the Versi Template Guide.

Get started with Versi in 10 minutes or less

Step 1. Start a project folder (1 min)

Yep, just create a folder.

Step 2. Add Versi (2 min)

Drop the contents of the “src” directory into your project folder. (Please note that Versi must be installed in the root directory of the web site. This will hopefully change in future releases, but is necessary for now.)

Step 3. Create an HTTP site (5 min)

  1. Point it to your project folder
  2. Point it to any database you want to work with

Step 4. Fire up your browser (2 min)

Point it to your HTTP site, and you’re done!

Next Steps

If you’ve successfully run your first page, and you don’t know what to do next, here are some suggestions:

Build

After you have added Versi to your project, you can start building pages by doing the following:

 

  1. Create a new page by copying one of the example pages.
  2. Add your params to the page by using lib-uitools (in the “lib” directory of Subversion) or some custom code.
  3. Build your display components in modules/display.xqy.
  4. Add your components to the page, and pass the page params to them.

 

For full documentation on building applications with Versi please see the Versi Template Guide.

Customize

Versi was built for your application and was built to be customized! Versi can be extended using css/appspecific.css and js/appspecific.js, because each page includes these files. Make upgrades easier by putting your code in these files so that when a new version of Versi is released you can drop the new code in, merge your files, and you’re ready to run!

Read

A helpful guide was put together to help you better understand Versi. If you’d like to find out more information about the template or its prebuilt widgets, consult the Versi Template Guide.

Play

Just like when we were kids, the best way to learn is to play! Check out the example pages provided and use your favorite editor to create new and interesting interfaces!

Collaborate

If you’re intrigued by what you’ve seen so far, please help make the project better by adding your own enhancements. This project was built to save you days of development time. Take a half-day or day to add a widget or some JavaScript, or your own creative vision!
